☁️ Синхронизация через GitHub Gist

Данные хранятся в приватном Gist твоего GitHub-аккаунта. Работает на любом устройстве — достаточно открыть этот файл и нажать «Загрузить».

Не настроена
КАК НАСТРОИТЬ — инструкция (нажми чтобы раскрыть)
1Перейди по ссылке → Создать токен выше. Откроется GitHub.
2В поле Expiration выбери No expiration. Галочка gist уже выставлена.
3Нажми Generate token. Скопируй токен — он начинается с ghp_.
4Вставь токен в поле выше и нажми Сохранить токен.
5Нажми ↑ Сохранить в облако — данные уйдут в GitHub. Gist ID заполнится сам.
6На другом устройстве открой этот файл, введи тот же токен и тот же Gist ID, нажми ↓ Загрузить из облака.
! Токен хранится только локально в этом браузере. Никогда не передавай его другим людям.
